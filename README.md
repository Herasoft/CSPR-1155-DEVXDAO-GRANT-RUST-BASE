# CSPR-1155-DEVXDAO-GRANT-RUST-BASE

Implementation of ERC1155 token for the CasperLabs platform.

## Install
Make sure `wasm32-unknown-unknown` is installed.
```bash
$ make prepare
```

## Build Smart Contract
```bash
$ make build-contract
```

## Test
Test logic and smart contract.
```bash
$ make test
```
